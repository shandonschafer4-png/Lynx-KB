Asset Rating

📅 Published June 2025 📂 Platform: Keytelemetrics 👤 Audience: Administrators

Asset Rating Profiles are used to rate the performance of an asset — such as a vehicle or driver — for each trip completed. A rating profile is built from events with assigned penalty values. When those events trigger during a trip, points are deducted from the score. The perfect rating score is 100.

Note: A client can have more than one Rating Profile, however an asset can only have one assigned to it at a time.

1. Creating a Rating Profile

Navigate to Administration → Profiles → Asset Rating, then follow these steps:

StepAction
1Click the + button above the Rating Profile list
2Enter a Name for the profile
3Click Next, confirm the name, then click Finish
Warning: The newly created profile will show a warning that it is incomplete until at least one penalty has been added.

2. Adding Events and Penalties

Edit the Rating Profile, then click Add to open the Rating Penalty box. Complete the fields and click OK, then Save.

FieldDescription
NameLabel for this penalty rule (e.g. Harsh Braking)
PenaltyPoints deducted per event occurrence
Max PenaltyMaximum total points that can be deducted from this single event type per trip
CostOptional cost value per event — used in Trip Analytics reports. Minimum value: 0.1
Event FilterThe specific event that triggers this penalty (e.g. Harsh Brake event)
Example: Harsh Braking penalty = 10 points, Max Penalty = 40. This means the driver can lose a maximum of 40 points from harsh braking per trip, regardless of how many times it triggers.

3. Understanding Negative Scores

The default Max Penalty per event is 100. If multiple penalties are added and Max Penalty is left at 100 for each, the minimum trip score can go into negative values. Use the table below to understand the difference:

ProfileEventsMax Penalty EachMinimum Score
Profile ISpeeding, Harsh Braking, Harsh Cornering100 each−200
Profile IISpeeding, Harsh Braking, Harsh Cornering20 / 40 / 400
Best practice: Set Max Penalty values so the combined total does not exceed 100 — this prevents negative scores and keeps ratings meaningful.

4. Assigning a Rating Profile to Assets

Individual Asset

Select the asset in Administration → Assets, open the Settings section, and assign the Rating Profile under Asset Rating.

Multiple Assets

Hold Ctrl and click to select multiple assets, then open the Settings section and assign the profile. All selected assets will be updated simultaneously.

Important: Any Rating Profile already assigned to a selected asset will be overwritten when assigning in bulk.

5. Viewing Rating Scores

The rating score displays as a coloured star and a percentage anywhere a trip appears. Hover over the star to reveal the exact score.

Icon ColourScore RangeMeaning
GREEN ★66 – 100%Good driver behaviour — few or no events triggered
ORANGE ★33 – 66%Average — some events recorded, review recommended
RED ★0 – 33%Poor — significant events recorded, discuss with driver

Rating scores appear in the following locations:

LocationWhat You See
Asset Overview → Asset DetailsLatest trip rating star for the selected asset
Trip Replay → Trip list per dayStar rating next to each trip entry
Trip Replay → Selected trip detailsRating %, penalty breakdown, event count, speed graph

6. Reporting on Asset Rating

Standard Reports

The Asset Rating Report shows the rating per trip or per day, including penalties received. The Trip Listing Report can also include the rating score column.

Analytics Reports / Dashboards

The Trip Analytics Report includes the Rating Profile as a parameter. Once run, Measures appear based on the Rating Profile configured for the client — including event counts (instances) and cost per event if assigned. Total event cost will appear in the graph when selected from the Measures panel.
