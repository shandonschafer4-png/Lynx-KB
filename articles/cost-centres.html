Cost Centres

📅 Published June 2025 📂 Platform: Keytelemetrics 👤 Audience: Administrators

Cost Centres were designed to restrict access and visibility between different divisions of the same company. Users and Assets are assigned to Cost Centres — users can only see, report on, and create alerts for assets within the same Cost Centres they belong to.

Applies to: Users  ·  Assets  ·  Locations  ·  Alerts  ·  Reports

1. How Cost Centres Work

Cost Centres follow a parent-child hierarchy. Unlike Asset Groups, there can only be one top-level parent Cost Centre to start. Children are added under this parent, and each child can itself become a parent.

The access rules work as follows:

User LevelAccess
City / Depot level userCan only see, report on, and create alerts for their own depot's assets
Regional / Branch managerCan see all depots within their region, but not other regions
Top management / Head officeCan see all assets across all Cost Centres regardless of region or depot
Tip: Plan your Cost Centre hierarchy to mirror your actual organisational structure — region → branch → depot works well for most fleet operations.

2. Creating Cost Centres

Step 1 — Rename the Default Cost Centre

Navigate to Administration → Organisation → Cost Centres. Select the Default Cost Centre, click Edit, rename it to your top-level division (e.g. "HQ" or "Main Branch"), and Save.

Step 2 — Add Child Cost Centres

Click the + button. The first child must be created under the renamed parent. After that, new Cost Centres can be added under the main parent or any existing child. Keep adding until your full structure is complete.

Example structure: HQ → East / North / West / South

3. Editing Cost Centre Information

Select any Cost Centre and click Edit to change its Name or Parent. You can also add Contacts — these appear to users on the Overview and Alerts screens with clickable email and phone links.

Contact FieldDescription
NameContact person's name
TypeRole type — Owner, Technical, Emergency, etc.
EmailClickable email link displayed on overview
MobileClickable mobile number on overview
PhoneLandline / office number
NotesAdditional context shown with the contact

4. Viewing Assets and Users in a Cost Centre

Select a Cost Centre from the list, then choose either the Assets or Users section in the right panel to see everything assigned to that Cost Centre.

Key difference from Asset Groups: Cost Centres control who can see what. Asset Groups control how assets are categorised. Both can be applied simultaneously.
